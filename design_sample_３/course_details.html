FEATURES

この講座の特徴

オイルトリートメント本講座が選ばれる、3つの理由

01

解剖学×実技の
完全連動カリキュラム

座学で学んだ知識をその週の実技で即実践。「わかる」が「できる」に変わるスピードが違います。

02

少人数制（定員6名）だから得られる、個別フィードバック

一人ひとりの施術を講師が直接確認。自分の癖や強みを把握しながら上達できます。

03

卒業後も続く、
伴走サポート

修了後もグループでの勉強会・個別相談を継続。開業・就職後の不安もTOMOSが一緒に解決します。

オイルトリートメントとは

オイルを使用してお客様の身体に直接触れる施術技術です。解剖学の知識をベースに、お客様一人ひとりの状態を見立て、最適なアプローチを選択する「オーダーメイドの施術力」を身につけます。

リラクゼーションセラピストとして、指名・リピートされ続けるための確かな技術と思いやりを同時に育てます。

なぜ一緒に解剖学を
学ぶのか

感覚だけの施術には限界があります。骨・筋肉・神経の仕組みを理解することで、お客様の状態を「見立てる力」が生まれます。TOMOSでは解剖生理学を技術と切り離さず、常に実技と連動させて学びます。

「なぜこのアプローチなのか」を自分の言葉で説明できるセラピストが、お客様から長く信頼され続けます。

POINT

このような方におすすめです

ひとつでも当てはまれば、この講座がきっとお役に立てます。

技術はあるのに、なぜか指名・リピートが取れないと悩んでいる方

感覚頼みの施術に限界を感じ、根拠のある技術を身につけたい方

解剖学を学びたいけど、どこから始めればいいかわからない方

卒業後も継続的にサポートしてくれる環境で学びたい方

独立・開業を視野に入れて、自信を持って施術できるようになりたい方

RESULT

受講前と受講後で、これだけの変化が生まれます。

CURRICULUM

カリキュラム詳細

何を、どの順番で学ぶのか。すべて公開しています。

座学

オンライン 1講座 2時間 計5回

身体の構造と理解・セラピストとしての判断力と応用力を身につけます

1 解剖学基礎

骨の役割と筋肉の役割

→ 骨と筋肉の役割

→ パーツ別解剖学基礎（骨盤と下肢編）

→ パーツ別解剖学基礎（上肢と体幹編）

2 姿勢の見立て

→ 拮抗筋とは？身体の連鎖について

→ 症状別で観るアプローチポイント

3 安心して施術するための
リスク管理

→ 症状別の判断基準

→ よくある疾患名や対応方法

4 カウンセリングスキル

→ リラクゼーションセラピストとしての心得

→ カウンセリングシートの活用方法

→ 選ばれるセラピストのカウンセリング思考

実技

オフライン 1講座 2時間 計9回（マスターコースは＋3回の計12回）

学んだ知識をもとに実践に落とし込み、施術の精度を高めます

1 身体のランドマークを
理解する

→ パーツ別ランドマーク（骨盤と下肢編）

→ パーツ別ランドマーク（上肢と体幹編）

2 体表解剖学
（描く／観る／触れる）

→ 体表解剖（表層筋編）

→ 体表解剖（深層筋編）

→ タオルワーク

→ 触り方、捉え方、手技の理解

4 コース別実技

→ 下肢編

→ 背中と上肢編

→ デコルテとヘッド編
